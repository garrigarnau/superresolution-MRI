Fase 1: Benchmarking

Avaluació de models pre-entrenats de super-resolució sobre imatges MRI sense fine-tuning

1. Objectiu

Avaluar com es comporten dos models de super-resolució "de sèrie" (sense adaptació al domini mèdic) quan s'apliquen a imatges de ressonància magnètica cerebral. Això estableix una baseline per comparar amb els models fine-tuned posteriorment.

2. Models Avaluats

Model A: Swin2SR

Transformer x4 HuggingFace

ID: caidas/swin2SR-classical-sr-x4-64
Arquitectura: Swin Transformer V2 adaptat per super-resolució
Entrenament original: Imatges naturals (DIV2K, Flickr2K)
Característiques: Mecanisme d'atenció que captura dependències a llarga distància. Tendeix a produir resultats suaus però consistents.

Model B: Real-ESRGAN

GAN x4 PyTorch

ID: ai-forever/Real-ESRGAN
Arquitectura: RRDB (Residual-in-Residual Dense Block) amb entrenament adversarial
Entrenament original: Imatges naturals amb degradacions realistes
Característiques: Genera detalls fins i textures gràcies al discriminador GAN. Pot introduir artefactes en dominis no vistos.

3. Pipeline d'Inferència

LR (64x64) Imatge test grayscale
→
RGB Conversion L → RGB (3 canals)
→
Model SR (x4) Swin2SR / ESRGAN
→
Grayscale RGB → L
→
SR (256x256) Resultat final
Nota: Ambdós models esperen imatges RGB (3 canals). Com que les nostres imatges MRI són en escala de grisos, les convertim a RGB abans de la inferència (duplicant el canal) i tornem a escala de grisos després.

4. Mètriques d'Avaluació

PSNR (Peak Signal-to-Noise Ratio)

Mesura la fidelitat píxel a píxel entre la imatge SR i la HR original. Valors més alts = millor reconstrucció. Unitats: dB.

PSNR = 10 * log10(MAX² / MSE)

SSIM (Structural Similarity Index)

Mesura la similitud perceptual considerant lluminositat, contrast i estructura. Rang: [0, 1], on 1 = idèntic.

SSIM(x,y) = f(luminancia, contrast, estructura)

Temps d'Inferència

Temps mitjà per processar una imatge (en segons). Mesurat amb torch.no_grad() per eliminar overhead de gradients.

5. Scripts

scripts/benchmark.py
Executa la inferència amb ambdós models sobre les 559 imatges del test set. Guarda les imatges SR a results/swin2sr/ i results/real_esrgan/. Registra el temps per imatge a results/timings.json.
scripts/evaluate.py
Calcula PSNR i SSIM comparant cada imatge SR amb la seva HR corresponent. Genera una taula comparativa i una graella visual (results/visual_comparison.png). Guarda mètriques a results/metrics.json.

Funcions principals de benchmark.py

run_swin2sr(lr_dir, output_dir)
Carrega el model Swin2SR des de HuggingFace amb AutoImageProcessor. Processa cada imatge LR amb el processador d'imatges del model i executa la inferència amb torch.no_grad().
run_real_esrgan(lr_dir, output_dir)
Carrega Real-ESRGAN amb pesos descarregats automàticament des de HuggingFace Hub. Utilitza el mètode predict() que gestiona internament el padding i el processament per patches.

Funcions principals de evaluate.py

compute_metrics(hr_dir, sr_dir)
Itera sobre totes les parelles HR/SR, calcula PSNR i SSIM amb scikit-image, i retorna les mitjanes i desviacions estàndard.
generate_visual_comparison(hr_dir, lr_dir, models, output_path)
Genera una imatge grid amb 5 mostres: LR (nearest-neighbor upscaled), HR ground truth, i les sortides de cada model. Permet avaluació visual ràpida.

6. Execució

# Activar l'entorn virtual
source venv/bin/activate

# 1. Executar la inferència (descarrega pesos automàticament)
python scripts/benchmark.py

# 2. Avaluar resultats
python scripts/evaluate.py

Anàlisi dels Resultats

Real-ESRGAN: Millor quantitativament

PSNR 8.26 dB superior i SSIM 0.19 punts més alt. Manté millor les intensitats originals i l'estructura general del cervell. Tot i això, introdueix artefactes d'over-sharpening i un aspecte "cartoonesc" a les textures cerebrals.

Swin2SR: Problemes greus de domini

El PSNR molt baix (15.40 dB) indica que el model altera significativament els valors d'intensitat. Produeix imatges rentades ("washed-out") amb pèrdua de contrast. El model d'atenció, entrenat en imatges naturals, no s'adapta bé al domini mèdic sense fine-tuning.

Velocitat

Swin2SR és 6.0× més ràpid que Real-ESRGAN (0.1037s vs 0.6192s per imatge). Real-ESRGAN processa per patches amb overlap, el que és més costós computacionalment.

Conclusió

Cap dels dos models funciona adequadament "de sèrie" per a imatges MRI. Ambdós van ser entrenats amb imatges naturals (fotografies) i no s'adapten al domini mèdic. Això justifica la necessitat de fine-tuning (Fase 2) per adaptar els models a les característiques específiques de les imatges de ressonància magnètica: baix contrast, escala de grisos, textures subtils del teixit cerebral.
