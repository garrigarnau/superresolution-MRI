2. Etapa 1 — Preprocessament

Entrada

Volums 3D de ressonància magnètica del dataset OASIS (Open Access Series of Imaging Studies). S'utilitzen els volums ja processats en espai Talairach, amb màscara cranial i correcció de biaix:

disc1/OAS1_XXXX_MR1/PROCESSED/MPRAGE/T88_111/*_masked_gfc.hdr

Procés

Volum 3D 176×208×176 vòxels
→
Slicing Axial 176 talls per eix Z
→
Filtratge >1% píxels cervell
→
Normalització min-max → [0, 255]
Resize → HR 256×256 (Bicúbic)
→
Downsample → LR 64×64 (Bicúbic ÷4)
→
Split 80/10/10 train/val/test
→
Guardar PNG data/{split}/HR/ i LR/

Detalls clau

  • Biblioteca: PIL (Pillow) amb interpolació Image.BICUBIC
  • Factor x4: No es fa SR en 3D. El volum es talla en slices 2D independents. Cada slice es redimensiona a 256² (HR), i després es redueix a 64² (LR). El model treballa amb imatges 2D.
  • Càrrega volums: nibabel (format Analyze .hdr/.img)
  • Seed: 42 per reproduïbilitat

3. Etapa 2 — Inferència (Benchmark)

Objectiu

Executar models pre-entrenats de super-resolució x4 sobre les imatges LR del test set i mesurar el temps d'inferència.

LR 64×64 Grayscale → RGB
→
Model SR ×4 Swin2SR / Real-ESRGAN
→
SR Output ~256×256 RGB
→
Postprocessat → Grayscale 256²

Swin2SR (Classical SR x4)

Transformer HuggingFace caidas/swin2SR-classical-sr-x4-64

  • Basat en Swin Transformer V2 adaptat a super-resolució
  • Entrada: imatge RGB processada amb AutoImageProcessor
  • Sortida: tensor reconstruction → es converteix a grayscale fent mitjana dels 3 canals
  • Si la sortida no és 256×256, es redimensiona amb bicúbic

Real-ESRGAN x4

GAN ai-forever RealESRGAN_x4.pth

  • Basat en ESRGAN amb entrenament en degradacions realistes
  • Entrada: imatge PIL RGB
  • Sortida: imatge PIL → .convert("L") per tornar a grayscale
  • Pesos descarregats automàticament a weights/RealESRGAN_x4.pth

Dispositiu d'inferència

Selecció automàtica: MPS (Apple Silicon) → CUDA → CPU. Configurable via --device.

FlagComportament
autoDetecta MPS/CUDA/CPU automàticament
gpuPrefereix MPS, si no CUDA
cudaForça CUDA (error si no disponible)
mpsForça MPS (Apple Silicon)
cpuForça CPU

4. Etapa 3 — Avaluació

Mètriques

Es comparen les imatges SR generades contra el Ground Truth (HR) amb dues mètriques estàndard:

PSNR (dB)

Peak Signal-to-Noise Ratio. Mesura la fidelitat píxel a píxel. Més alt = millor.

SSIM

Structural Similarity Index. Mesura la similitud perceptual (estructura, lluminositat, contrast). Rang [0, 1].

Biblioteca: scikit-image (skimage.metrics)

Comparació visual

Es genera una graella d'imatges amb columnes: LR (64×64, nearest-neighbor upscaled), HR (Ground Truth), i la sortida SR de cada model. Es guarda com a results/visual_comparison.png.

Pipeline d'avaluació

Carregar HR data/test/HR/*.png
→
Carregar SR results/{model}/*.png
→
Resize si cal Match HR dimensions
→
PSNR + SSIM Per imatge → mitjana

S'autodetecten tots els directoris de resultats disponibles (inclosos *_postprocessed).

6. Execució Completa

# 1. Setup
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# 2. Preprocessament (genera data/train|val|test/HR|LR/)
python scripts/preprocess.py

# 3. Inferència dels models SR (genera results/swin2sr/ i results/real_esrgan/)
python scripts/benchmark.py --device auto

# 4. Avaluació (genera metrics.json i visual_comparison.png)
python scripts/evaluate.py
