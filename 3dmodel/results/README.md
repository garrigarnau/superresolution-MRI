# 3D Super-Resolution Results

This folder contains lightweight, GitHub-friendly outputs from the 3D MRI
super-resolution experiment. Large generated volumes, patches, checkpoints, and
the original OASIS data are intentionally not tracked in Git.

## Test Metrics

Evaluation was run on 7 OASIS test subjects using 3D PSNR and 3D SSIM.

| Method | PSNR (dB) | SSIM |
|---|---:|---:|
| MedicalNet SR 3D | 33.08 +/- 1.00 | 0.9164 +/- 0.0181 |
| Trilinear baseline | 33.10 +/- 0.76 | 0.9411 +/- 0.0044 |

The residual 3D model now reconstructs anatomy at approximately trilinear
quality. It slightly underperforms trilinear on mean PSNR (-0.02 dB) and SSIM
(-0.0247), so it is a functional baseline rather than a clear improvement.

## Files

- `metrics_3d.json`: full per-subject metrics and summary statistics.
- `figures/metrics_summary.png`: PSNR and SSIM summary plot.
- `figures/OAS1_0002_MR1_comparison.png`: central-slice qualitative comparison.

## Reproducibility

The full local outputs are generated by:

```bash
cd 3dmodel
python preprocess.py
python test_pipeline.py --gpu
python train.py
python infer.py
python evaluate.py
```

Generated heavy files are ignored by Git:

- `3dmodel/patches/`
- `3dmodel/checkpoints/`
- `3dmodel/results/**/*.npy`
- `disc1/`
